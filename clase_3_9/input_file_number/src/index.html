INPUT FILE

Suba archivos por medio de un formulario.

Input File es un elemento de entre los más viejos en existencia en HTML. Consiste en la idea que un usuario puede elegir y subir uno o múltiples archivos por medio de un formulario a un destino determinado.

El proceder es: El elemento file abre un elemento para elegir uno (o múltiples, si se incluye el atributo Multiple dentro del picker, y de acuerdo a esos archivos y el botón "Submit" procede a activar el proceso seleccionado por el "post"del formulario.

 

Atributos modificables

  • Multiple como parametro para múltiples archivos. (No agregar si es solo uno)
  • Size - Seguido por valor numérico integer
