INPUT FILE

Suba archivos por medio de un formulario.

Input File es un elemento de entre los más viejos en existencia en HTML. Consiste en la idea que un usuario puede elegir y subir uno o múltiples archivos por medio de un formulario a un destino determinado.

El proceder es: El elemento file abre un elemento para elegir uno (o múltiples, si se incluye el atributo Multiple dentro del picker, y de acuerdo a esos archivos y el botón "Submit" procede a activar el proceso seleccionado por el "post"del formulario.

 

Atributos modificables

  • Multiple como parametro para múltiples archivos. (No agregar si es solo uno)
  • Size - Seguido por valor numérico integer
  • Accept - Qué tipos de archivos está dispuesto a aceptar el formulario (accept="file_extension|audio/*|video/*|image/*|media_type"). Acepta todos los tipos de archivo reconocidos por IANA.

INPUT NUMBER

Haga operaciones numéricas por medio de un formulario.

Input Number consiste en un input de tipo numérico, sea con un principio y un fin establecido, o un valor libre. (Cualquier elemento es válido siempre y cuando sea un elemento numérico.)

Atributos modificables

  • Min como valor numérico mínimo.
  • Max como valor numérico máximo.
  • Step - La propiedad step permite el aumento o disminución de un número incrementalmente por valores (por ejemplo, un valor step="10" permitiría agregar o quitar números de a 10 por vez.)
